Short answer

A public RPC endpoint (e.g., QuickNode, Alchemy, Helium, or the free Solana “mainnet‑beta” endpoint).
A Python Solana client library (solana‑py or anchorpy) that talks to the RPC.
Client‑side wallet integration (Phantom, Solflare, etc.) using the standard Solana‑Web3‑JS library or the wallet‑adapter package.

## 5️⃣ Flask backend – core logic

Below is a **complete, minimal Flask app** that:

* Issues a nonce.
* Verifies the signed message.
* Queries the Solana RPC for ownership of a given **NFT mint** *or* checks a generic SPL token balance.
* Grants/denies access to a protected route.

```python
# app.py
import os
import json
import secrets
from flask import Flask, request, jsonify, session, redirect, url_for
from solana.rpc.api import Client
from solana.publickey import PublicKey
from solana.transaction import Transaction
from solana.rpc.async_api import AsyncClient
from solana.rpc.commitment import Confirmed
from base64 import b64decode

app = Flask(__name__)
app.secret_key = os.getenv("FLASK_SECRET_KEY", secrets.token_hex(32))

# -------------------------------------------------
# 1️⃣ RPC configuration – pick any public endpoint
# -------------------------------------------------
SOLANA_RPC_URL = os.getenv(
    "SOLANA_RPC_URL",
    "https://solana-mainnet.g.alchemy.com/v2/your-api-key"  # <-- replace with your own
)
rpc = Client(SOLANA_RPC_URL)

# -------------------------------------------------
# 2️⃣ Config – the NFT / SPL token you gate with
# -------------------------------------------------
GATED_NFT_MINT = PublicKey("YOUR_NFT_MINT_ADDRESS_HERE")          # e.g. 9xQ... (32‑byte base58)
# If you want to gate by a generic SPL token (e.g., $MYTOKEN)
GATED_SPL_MINT = PublicKey("YOUR_SPL_TOKEN_MINT_ADDRESS_HERE")
MIN_SPL_BALANCE = 1          # at least 1 token (for NFTs this will be 1)

# -------------------------------------------------
# Helper: verify Solana signed message (ed25519)
# -------------------------------------------------
def verify_signature(pubkey_str: str, signature_hex: str, message: str) -> bool:
    try:
        from nacl.signing import VerifyKey
        from nacl.exceptions import BadSignatureError

        verify_key = VerifyKey(bytes(PublicKey(pubkey_str)))
        signature = bytes.fromhex(signature_hex)
        verify_key.verify(message.encode(), signature)
        return True
    except BadSignatureError:
        return False
    except Exception as e:
        print("Verification error:", e)
        return False

# -------------------------------------------------
# 3️⃣ Nonce endpoint (GET)
# -------------------------------------------------
@app.route("/auth/nonce", methods=["GET"])
def get_nonce():
    nonce = secrets.token_urlsafe(16)
    session["nonce"] = nonce
    return jsonify({"nonce": nonce})

# -------------------------------------------------
# 4️⃣ Verify endpoint (POST) – core gating logic
# -------------------------------------------------
@app.route("/auth/verify", methods=["POST"])
def verify():
    data = request.get_json()
    address = data.get("address")
    signature = data.get("signature")
    nonce = data.get("nonce")

    # Basic sanity
    if not all([address, signature, nonce]):
        return jsonify({"allowed": False, "error": "missing fields"}), 400

    # Verify nonce matches session (prevents replay)
    if nonce != session.get("nonce"):
        return jsonify({"allowed": False, "error": "invalid nonce"}), 400

    # 1️⃣ Verify signature
    if not verify_signature(address, signature, nonce):
        return jsonify({"allowed": False, "error": "bad signature"}), 400

    # 2️⃣ Query RPC for token accounts owned by address
    owner_pubkey = PublicKey(address)

    # -------------------------------------------------
    #   a) Check NFT ownership (exact mint match)
    # -------------------------------------------------
    try:
        nft_accounts = rpc.get_token_accounts_by_owner(
            owner_pubkey,
            mint=GATED_NFT_MINT,
            encoding="jsonParsed",
        )

        nft_owned = False
        for acct in nft_accounts["result"]["value"]:
            amount = int(acct["account"]["data"]["parsed"]["info"]["tokenAmount"]["amount"])
            if amount > 0:
                nft_owned = True
                break

        if nft_owned:
            session["allowed"] = True
            return jsonify({"allowed": True, "method": "nft"})
    except Exception as e:
        print("NFT query error:", e)

    # -------------------------------------------------
    #   b) Fallback: generic SPL token balance check
    # -------------------------------------------------
    try:
        spl_accounts = rpc.get_token_accounts_by_owner(
            owner_pubkey,
            mint=GATED_SPL_MINT,
            encoding="jsonParsed",
        )

        spl_balance = 0
        for acct in spl_accounts["result"]["value"]:
            info = acct["account"]["data"]["parsed"]["info"]
            bal = int(info["tokenAmount"]["amount"])
            spl_balance += bal

        if spl_balance >= MIN_SPL_BALANCE:
            session["allowed"] = True
            return jsonify({"allowed": True, "method": "spl", "balance": spl_balance})
    except Exception as e:
        print("SPL query error:", e)

    # -------------------------------------------------
    #   c) Not authorized
    # -------------------------------------------------
    session["allowed"] = False
    return jsonify({"allowed": False, "error": "no token found"}), 403

# -------------------------------------------------
# 5️⃣ Protected route – only reachable after auth
# -------------------------------------------------
@app.route("/protected")
def protected():
    if not session.get("allowed"):
        return redirect(url_for("get_nonce"))  # or render a “please log in” page
    return """
    <h1>🔐 Welcome, token‑holder!</h1>
    <p>This is your gated content.</p>
    """

# -------------------------------------------------
# 6️⃣ Simple home page (optional)
# -------------------------------------------------
@app.route("/")
def index():
    return """
    <h2>Flask + Solana NFT gate demo</h2>
    <button onclick="connectAndAuth()">Connect wallet → enter app</button>
    <script src="/static/connect.js"></script>
    """

if __name__ == "__main__":
    app.run(host="0.0.0.0", port=5000, debug=True)
```

## 6️⃣ Optional: Use a **3rd‑party API** to avoid any RPC code

If you truly want **zero Solana‑specific code** in your Flask backend, there are services that expose an HTTP endpoint like:

```
GET https://api.nftport.xyz/v0/check_ownership?owner_address=...&contract_address=...
Headers: Authorization: <YOUR_NFTPORT_API_KEY>
```

- **NFTPort**, **Moralis**, **QuickNode “NFT API”**, **Helium “Token API”**.
- They return a simple JSON `owns: true/false` (or the token balance) which you can forward to your Flask gate.
- **Pros:** No `solana-py` library, no need to parse RPC responses.
- **Cons:** You’re dependent on the provider’s rate limits and uptime; you also give them the user’s public address (which is already public).

You can replace the RPC‑query block in the Flask example with a call to one of those services if you want to keep the code even slimmer.

| **Cache RPC responses** (reduce latency) | Use `functools.lru_cache` on the RPC query functions or a Redis store. |

| **Deploy on a serverless platform** (AWS Lambda, Cloud Run) | Wrap the Flask app with `Zappa` or `Serverless WSGI`. No changes to the token‑gating logic. |
| **Add a “logout”** | Clear the session: `session.clear()` and redirect to home. |

All of the above are **pure Python/JavaScript** – you never need to compile or run Rust code.
